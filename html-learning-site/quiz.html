ÔøΩ Quiz Answer Key (Don't peek until you're done!)
Correct Answers & Explanations
Question Correct Answer Why It's Right
1. What does HTML stand for? HyperText Markup Language It's the standard language for creating web pages. "HyperText" refers to links, and "Markup" refers to tags.
2. Biggest heading? <h1> Headings are ranked from <h1> (most important) to <h6> (least important).
3. Correct image syntax? <img src="image.jpg" alt="My Image"> The src attribute defines the image source, and the alt attribute provides essential alternative text.
4. Numbered list tag? <ol> <ol> stands for "Ordered List," which is numbered. <ul> is for bulleted "Unordered Lists."
5. Table row tag? <tr> <tr> stands for "Table Row." It contains <th> (header) or <td> (data) cells.
6. Semantic tag for "important"? <strong> While <b> makes text bold, <strong> tells browsers and screen readers the text is important.
7. Dropdown list tag? <select> The <select> tag is used to create a dropdown list, with <option> tags inside it.
8. Input for multiple options? type="checkbox" Checkboxes allow for zero, one, or multiple selections. Radio buttons only allow one.
9. How to embed a YouTube video? Using an <iframe> YouTube provides a ready-to-use <iframe> embed code for easily placing videos on your site.
10. Purpose of `alt` attribute? To provide alternative text for screen readers This is a critical accessibility feature for visually impaired users and also displays if the image fails to load.
11. Most semantic structure? <header>, <article>, <footer> These HTML5 tags describe the meaning of the content, which is better for SEO and accessibility than generic <div> tags.
12. Purpose of `<label>`? To link a text description to a form control This improves user experience (you can click the label to focus the input) and is essential for screen readers.
